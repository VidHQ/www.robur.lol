Module common.userinput

common userinput recorder and playback module.

Can be used to make keybindings or send keystrokes.

EVENTs:

KEYUP             useful for keybindings
MOUSE             Windows mouse events. check MSDN.
KEYBOARD          Windows keyboard events. check MSDN.
MOUSE_WHEEL_UP    Self explaining
MOUSE_WHEEL_DOWN  Self explaining

Examples:

register("KEYUP", function(e) printf("key pressed: " .. e.char .. ". replace me with your function!") end)
