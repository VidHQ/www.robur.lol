Module common.asmcall

This module provides functions to call any kind of assembler functions..

  • thread synchronization
  • register setup
  • calling conventions

Examples

-- on Windows call MessageBoxA popup with 'Hello World!'
asmcall.cdecl(getProcAddress("user32", "MessageBoxA"), 0, "Hello World!", "Title", 0)

-- call address 0x123456 with eax set to 1 and ecx to something else
asmcall.fastcall(0x123456, {eax=1, ecx=0xdeadbeef}, arg1, ...)

-- call address 0x123456 without any register setup in plain __cdecl fashion
asmcall.cdecl(0x123456, arg1, ...)

-- call a 64bit function using the w64 calling convention (only in 64bit process)
asmcall.w64(<address>, arg1, ...)

32bit fastcall call via the current thread.

Usage:

    fastcall(0x123456, {eax=1, ecx=0xdeadbeef}, arg1, ...)
cdecl (address, ...)
Shorthandle to fastcall So it's actually a __cdecl calling convention
stdcall (address, ...)
The stdcall convention means the callee cleans up stack arguments.

Our current implementation doesn't care for that, and it's safe to simply use cdecl instead! (This function is just an alias name.)

w64 (address, ...)
This will make a call using the Windows 64bit calling convention.
