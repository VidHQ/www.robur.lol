Module common.log

common logging module.

Functions

setup (module)
helper function to create local logging shortcuts (DEBUG, INFO, ...) for a Lua module. It is intended to be called as a 'decorator' function (parameter to module). The resulting proxy functions will nicely forward your messages to the log system, with the module's _NAME field indicating their origin.

Usage:

    require("common.log")
module("modname", package.seeall, log.setup, ...)
clean.module("modname", clean.seeall, log.setup, ...)
LOG (level, mod, fmt, ...)
Logger main function.

Any log message has to pass through this function. Message output will be suppressed if level is below the current log level! (E.g. for normal operation, you won't get any LOG_DEBUG messages.) Error logging also automatically causes a stacktrace.

See also:

    log.setLevel
